# SectionHead

Use this to alert the user of an event.

<Canvas>
  <Story name="Component">
    <SectionHead
      heading="This is a h2 heading."
      headingTag="h2"
      subheading="This is some subheading copy."
      subheadingTag="p"
    />
  </Story>
</Canvas>

## Alignment

Three alignments are supported.

<Canvas>
  <Story name="Alignment">
    <SectionHead
      heading="Left aligned"
      headingTag="h2"
      subheading="This is some subheading copy."
      alignment="left"
    />
    <SectionHead
      heading="Center aligned"
      headingTag="h2"
      subheading="This is some subheading copy."
      alignment="center"
    />
    <SectionHead
      heading="Right aligned"
      headingTag="h2"
      subheading="This is some subheading copy."
      alignment="right"
    />
  </Story>
</Canvas>
